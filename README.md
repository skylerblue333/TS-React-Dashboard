# TS-React-Dashboard

![CI](https://github.com/skylerblue333/TS-React-Dashboard/workflows/CI/badge.svg)

Production-ready TypeScript Express API for dashboard operations.

## Features
- Fully typed Express handlers
- Supertest integration testing
- Docker containerization
- GitHub Actions CI Pipeline

## Quick Start
```bash
npm ci
npm test
npm run build && npm start
```
